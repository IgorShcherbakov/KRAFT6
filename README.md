## Подготовка 
### 1. Создать виртуальное окружение
```bash
python -m venv venv
```
### 2. Активировать виртуальное окружение
```bash
venv\Scripts\Activate.ps1
```
### 3. Установить зависимости
```bash
pip install -r requirements.txt
```
### 4. Поднять контейнеры
```bash
docker compose up -d
```
